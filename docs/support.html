How to Get Help

For bug reports, questions, or feature requests, use the GitHub repository for the website/support pages (or your extension repo if you choose to move support there later).

Open GitHub Issues Repository

What to Include in a Report

  • Browser version (Microsoft Edge version number)
  • Extension version (shown in the Options page badge)
  • What site/page you were on (domain is enough if private)
  • What you expected vs what happened
  • Screenshots if relevant
