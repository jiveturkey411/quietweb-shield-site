QuietWeb Shield

Home Support

Effective Date

February 22, 2026

Summary

QuietWeb Shield is designed to block advertising and common tracking network requests locally in your browser. The extension does not require an account and does not send browsing history or page content to a developer-operated server.

Information the Extension Accesses

TypePurpose
Active tab URL / domain (local use) Used in the popup to detect the current site's domain so you can add/remove that domain from the allowlist.
Extension settings and allowlist Stored using browser extension storage (chrome.storage.sync / chrome.storage.local) to preserve your preferences.
Filter list downloads The extension downloads filter lists from EasyList / EasyPrivacy sources to refresh blocking rules.
Diagnostics counters (local) Blocked request counts and update timestamps may be stored locally to display status in the extension UI.

What Is Not Collected or Sold

  • No account registration data.
  • No sale of personal information.
  • No developer analytics/telemetry endpoint receiving your browsing activity.
  • No remote code execution service or cloud profile service.

Third-Party Services

QuietWeb Shield retrieves filter list content from publicly available list providers, including:

  • EasyList
  • EasyPrivacy

Requests to those providers are subject to their own policies and infrastructure logging practices.
