QuietWeb Shield

Manifest V3 ad and tracker blocker for Microsoft Edge using declarative network request rules and curated EasyList + EasyPrivacy sources.

Privacy Policy Support GitHub

What It Does

  • Blocks ad and tracker requests at the network level.
  • Uses Manifest V3 declarativeNetRequest rules.
  • Supports daily filter refresh plus manual refresh.
  • Supports per-domain allowlist controls.

What It Does Not Do (v1)

  • No cosmetic filtering/scriptlets.
  • No account/login system.
  • No backend sync service operated by the developer.
  • No telemetry collection by the extension developer.
